

# __Installation__
### [Arch based linux](#arch)
### [Ubuntu based linux](#ubuntu)


### __Arch__
Python 3.9
```
$ pacman -S python3
```
You also require to have Scapy and Shodan modules installed
```
$ pip install Scapy
$ pip install Shodan
```
### __Ubuntu__
Python 3.9
```
$ apt install python3
```
You also require to have Scapy and Shodan modules installed

```
$ pip install Scapy
$ pip install Shodan
```

Using Shodan API
This tool requires you to own an upgraded Shodan API
